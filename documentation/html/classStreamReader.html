Detailed Description

Base StreamReader abstract class. All user-implemented stream readers must inherit from this.

This provides an istream* member and a constructor taking an istream object.

All derived classes must implement a templated `load()' function which reads data into the given variable, and it is the correct place to do type-checking if necessary.

For classes, where user-intervention is required, `load()' can depend on the user-defined `deserialize()' function for that class which will perform the deserialization for individual members.
